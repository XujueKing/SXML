# SXML Framework - Clean Template

一个干净的 SXML 框架模板，用于快速启动新项目。

## 快速开始

1. 复制此目录到你的项目位置
2. 运行 `npm install` 安装依赖
3. 运行 `node dev-server-sxml.js dev` 启动开发服务器
4. 访问 http://localhost:3000

## 包含内容

-  完整的 SXML 框架（编译器 + 运行时）
-  开发服务器（支持实时编译和 API 代理）
-  构建工具（支持 dev/test/prod 环境）
-  示例页面（pages/index）
-  完整文档（docs 目录）

## 不包含内容

-  业务页面（login、webapp 等）
-  业务逻辑和数据
-  特定项目的配置
